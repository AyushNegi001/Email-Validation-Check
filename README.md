A simple Email Validation project built using Python. This is one of my beginner Python projects. The program checks whether an email address is valid or invalid using string methods, loops, and conditional statements without using any external libraries or regular expressions.

🧠 How It Works

User enters an email address
Checks the minimum length of the email
Verifies that the first character is an alphabet
Ensures exactly one @ symbol is present
Validates the position of the . in the domain
Detects spaces and invalid characters
Displays whether the email is valid or invalid

📧 Validation Rules

Email length must be at least 15 characters
First character must be an alphabet
Only one @ symbol is allowed
Dot (.) must be in the correct position
Spaces are not allowed
Only letters, digits, _, ., and @ are allowed

🚀 Features

String-based email validation
User input system
Character-by-character checking
Invalid character detection
Beginner-friendly Python project
No external libraries required

🛠️ Technologies Used

Python
String Methods
Loops
Conditional Statements
